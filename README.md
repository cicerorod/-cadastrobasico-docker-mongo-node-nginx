## ![](https://img.icons8.com/metro/20/000000/run-command.png) Execução

1. Clonar via prompt de comando o projeto em uma pasta de sua preferência: `git clone https://github.com/cicerorod/cadastrobasico-docker-mongo-node-nginx.git`
2. Acessar a pasta principal do projeto via prompt de comando. Ex: `cd cadastrobasico-docker-mongo-node-nginx`;
3. Executar o comando `docker-compose up` criar os containers . Ex: `docker-compose up`;
4. Acessar o endereço `http://localhost:3001/`
